         Creates a JWT access token with claims: sub=userID.String(), exp=30 days from now, iat=now.
         Signs with HS256 using s.jwtSecret.

         Generates 32 random bytes, base64url-encodes with "qrf_" prefix.
         Hashes with token.HashToken(plaintext).
         Stores hash in DB via s.queries.CreateRefreshToken with expires_at = 90 days.
         Returns plaintext.

       - func (s *AuthService) RefreshSession(ctx, refreshPlaintext string) (newAccessToken, newRefreshPlaintext string, err error)
         Hashes the plaintext, looks up in DB via GetRefreshToken.
         If not found or expired, return error.
         Revoke old token via RevokeRefreshToken.
         Create new access token via CreateSession.
         Create new refresh token via CreateRefreshToken.
         Return both. (Rotate-on-use per research recommendation)

       - func (s *AuthService) Logout(ctx, refreshPlaintext string) error

    2. Create relay/internal/middleware/jwtauth.go:
       ```go
       package middleware

       import (
           "context"
           "net/http"

           "github.com/go-chi/jwtauth/v5"
       )

       type userIDKey struct{}

       // NewJWTAuth creates a jwtauth.JWTAuth instance for the given secret.
       func NewJWTAuth(secret string) *jwtauth.JWTAuth {
           return jwtauth.New("HS256", []byte(secret), nil)
       }

       // UserIDFromContext extracts the user ID string from the JWT claims in context.
       func UserIDFromContext(ctx context.Context) (string, bool) {
           _, claims, err := jwtauth.FromContext(ctx)
           if err != nil {
               return "", false
           }
           sub, ok := claims["sub"].(string)
           return sub, ok
       }
       ```
       This wraps go-chi/jwtauth/v5. The Verifier and Authenticator middlewares from jwtauth are used directly in the router (not wrapped here).

       - Define AuthHandler struct with fields: svc *service.AuthService, googleConfig *oauth2.Config, githubConfig *oauth2.Config, frontendURL string
       - func NewAuthHandler(svc, googleConfig, githubConfig, frontendURL) *AuthHandler

       - func (h *AuthHandler) GoogleLogin(w, r):
         Generate 32-byte random state string (crypto/rand, hex-encoded).
         Set httpOnly cookie "oauth_state" with value=state, MaxAge=300 (5 min), Secure=true, SameSite=Lax (NOT Strict — Pitfall 4), Path="/".
         Redirect to h.googleConfig.AuthCodeURL(state).

       - func (h *AuthHandler) GoogleCallback(w, r):
         Read "oauth_state" cookie, compare to r.URL.Query().Get("state"). If mismatch, return 400.
         Exchange code: h.googleConfig.Exchange(ctx, r.URL.Query().Get("code")).
         Fetch Google userinfo from https://www.googleapis.com/oauth2/v2/userinfo with the token.
         Parse email, name, picture from JSON response.
         Call h.svc.UpsertOAuthUser(ctx, email, name, picture, "google", googleID).
         Read anon_sid cookie, call h.svc.ClaimAnonymousRooms(ctx, user.ID, anonSID).
         Create session: accessToken, _ := h.svc.CreateSession(user.ID).
         Create refresh token: refreshToken, _ := h.svc.CreateRefreshToken(ctx, user.ID).
         Set httpOnly cookies:
           - "jwt": value=accessToken, MaxAge=30*24*3600, HttpOnly=true, Secure=true, SameSite=Lax, Path="/"
           - "refresh_token": value=refreshToken, MaxAge=90*24*3600, HttpOnly=true, Secure=true, SameSite=Lax, Path="/auth/refresh"
         Clear the "oauth_state" cookie (MaxAge=-1).
         Redirect to h.frontendURL + "/dashboard".

       - func (h *AuthHandler) GitHubLogin(w, r):
         Same pattern as GoogleLogin but using h.githubConfig.

       - func (h *AuthHandler) GitHubCallback(w, r):
         Same pattern as GoogleCallback but:
         - Exchange code with h.githubConfig
         - Fetch GitHub user from https://api.github.com/user with the token (Authorization: token <access_token>)
         - Also fetch email from https://api.github.com/user/emails (GitHub may not include email in user endpoint)
         - Parse login, name, avatar_url, email from response
         - Provider is "github", providerID is the GitHub user ID (strconv.Itoa from the id field)

       - func (h *AuthHandler) Logout(w, r):
         Read "refresh_token" cookie, call h.svc.Logout(ctx, refreshToken).
         Clear "jwt" cookie (MaxAge=-1).
         Clear "refresh_token" cookie (MaxAge=-1).
         Return 200 {"message": "logged out"}.

       - func (h *AuthHandler) RefreshToken(w, r):
         Read "refresh_token" cookie.
         Call h.svc.RefreshSession(ctx, refreshToken).
         Set new "jwt" and "refresh_token" cookies (same attributes as in callback).
         Return 200 {"message": "session refreshed"}.

       - func (h *AuthHandler) Me(w, r):
         Extract user ID from JWT claims via middleware.UserIDFromContext(r.Context()).
         Parse UUID, call queries.GetUserByID.
         Return user info (email, display_name, avatar_url, provider).
