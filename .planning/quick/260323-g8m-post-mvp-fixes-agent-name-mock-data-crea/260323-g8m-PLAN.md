**Fix 1: Buffer the hub broadcast channel (hub.go line 85)**

In `NewRoomHub`, change:
```go
broadcast: make(chan broadcastCmd),
```
to:

This matches the pattern already used for `subscribe` (which sends to `resp chan error, 1`) and makes broadcast non-blocking when the hub goroutine is briefly busy. Size 64 matches the per-subscriber channel size used in `Subscribe()`.

**Fix 3: Extract agent_name from Authorization header (handler.go)**

The A2A spec sends the bearer token in `Authorization: Bearer <token>`. The agent's name is stored in the PresenceRegistry when they JOIN (via `card.Name`). To get the name at message/send time:

1. Add `registry *hub.PresenceRegistry` to `handleA2ARequest` and `handleMessageSend` signatures. Update `MountA2ARoutes` to pass registry through.

2. In `handleMessageSend`, extract bearer token from header:
```go
agentName := params.Message.Role // fallback
authHeader := r.Header.Get("Authorization")
if strings.HasPrefix(authHeader, "Bearer ") {
    bearerToken := strings.TrimPrefix(authHeader, "Bearer ")
    // Look up the agent by token in registry by scanning registered agents
    // Registry.GetByToken does not exist yet — use Registry.List(roomID) and match
    agents := registry.List(roomID)
    for _, a := range agents {
        if token.VerifyToken(bearerToken, room.TokenHash) {
            // All agents in a room share the room bearer token.
            // We cannot distinguish agents by token alone.
            // Use X-Agent-Name header as the agent identifier.
            break
        }
    }
}
```

NOTE: All agents in a room share the same room bearer token (per Quorum's architecture — one token per room, not per agent). So we cannot identify individual agents by token. The correct approach is:

**Use `X-Agent-Name` header.** Agents that called `/r/{slug}/join` sent their card.Name. The message/send handler should read `X-Agent-Name` header as the agent identifier, falling back to `params.Message.Role` if absent.

Implementation:
```go
// Extract agent name: prefer X-Agent-Name header (set by agents that joined),
// fall back to message role field.
agentName := r.Header.Get("X-Agent-Name")
if agentName == "" {
    agentName = params.Message.Role
}
if agentName == "" {
    agentName = "agent"
}
```

Then pass `agentName` to `messages.Append(roomID, agentName, textContent)` replacing `params.Message.Role`.

Also add `"strings"` to imports in handler.go if not already present (needed for `strings.HasPrefix` / `strings.TrimPrefix` pattern — even if not used for token lookup, keep import clean).

**Wire everything:**
- `MountA2ARoutes` signature: add `registry *hub.PresenceRegistry` parameter
- Update call site in `cmd/server/main.go` (or wherever MountA2ARoutes is called) to pass registry
- `handleA2ARequest` receives `hubMgr` and `registry`, passes both to `handleMessageSend`
- `handleMessageSend` receives `hubMgr` and `registry` (registry not used in first cut, but wire it for future)

Check `cmd/server/main.go` for MountA2ARoutes call site and update accordingly.
